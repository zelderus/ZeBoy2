ZeBOY 2
Home LCD SRAM EEPROM C Program Release

Написание приложения на языке 'C' в среде 'Keil' под MK.

Микроконтроллер. Даташит AT89C51RC-24PU фирмы Atmel.

IDE Keil.

Цель

Написание программы для MK AT89C51RC на языке уровнем повыше, таком как C.

Создание проекта

1. Project - New Project
2. Указать папку и название проекта.
3. Выбрать левайс. В нашем случае AT89C51RC.

Настройка проекта

1. Правой кнопкой по Target1 - Options for Target1.
2. Вкладка Target => Memory Model: Large var in XDATA
3. Вкладка Output => Create HEX file

Программа

Программа-игра на C можно взять с GIT.

Компиляция

Project - Rebuild all target files
В папке Objects (поумолчанию) появятся файлы. В том числе файл .hex (если не забыли указать в настройках проекта).
Этот файл понадобится для конвертации в BIN и записи его программатором на МК, а также для тестирования на Proteus.
